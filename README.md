# customer purchase behaviour
This project builds a machine learning classification model to predict customer purchase categories based on behavioral and demographic data.

## Dataset
The Dataset contains customer information such as age, income, monthly spending, session duration, page views per visit, device type and membership tier.

## Approach
- Data cleaning and handling missing values
- Exploratory Data Analysis(EDA)
- Feature encoding and scaling
- Model training using DecisionTreeClassifier
- model evaluation using accuracy_score and cross_validation

## Tools & Techniques
python
pandas, numpy
matplotlib
scikit_learn

## Results
The model was evaluated using accuracy to assess performence
